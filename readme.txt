네이버 블로그와 워드프레스를 동기화

== Description ==

네이버 블로그와 워드프레스를 동기화 해주는 플러그인 입니다.
본 플러그인은 php namespace 를 사용하므로 php버전이 5.3 미만인 경우 동작하지 않습니다.

1. [쓰기] 워드프레스 블로그에서 작성한글이 네이버 블로그에도 동일하게 적용되어 등록됩니다. 원하는 카테고리 선택은 물론 등록여부를 결정할수도 있도록 만들어저 있습니다.
2. [수정] 작성한글을 수정할경우 네이버 블로그에도 동일하게 수정되어 적용됩니다.
3. [삭제] 작성한글이 삭제될경우 네이버 블로그에 등록된 글이 동일하게 삭제됩니다.

== Installation ==

1. 플러그인 활성화
2. 네이버 블로그 접속 -> 내메뉴 -> 관리 -> 메뉴글관리 -> 글쓰기API설정에서 API연결암호 확인
3. 워드프레스 관리자화면 ) 설정 -> IdeaNaverSync 에서 각 설정항목 입력. 입력이 완료된후 연결검사
